# 📝 NOTIZBLOCK

Ein einfacher und funktionaler Notizblock, den ich während meiner Weiterbildung entwickelt habe.  
Das Projekt wurde erstellt, um grundlegende Programmierkenntnisse praktisch anzuwenden und den Umgang mit Benutzeroberflächen, 
Dateioperationen und Anwendungslogik zu vertiefen.

---

## 🚀 Funktionen

- Erstellen von Textnotizen
- Speichern von Dateien
- Öffnen vorhandener Dateien
- Übersichtliche Benutzeroberfläche
- Einfache Bedienung

---

## 🛠️ Verwendete Technologien

- HTML / CSS / JavaScript
- GitHub

---

## 🎯 Ziel des Projekts

Das Ziel dieses Projekts war es, praktische Erfahrungen in der Softwareentwicklung zu sammeln 
und ein funktionierendes Desktop-Programm eigenständig umzusetzen.

Während der Weiterbildung konnte ich dadurch wichtige Grundlagen lernen, darunter:

- Programmstruktur und Logik
- Arbeiten mit Dateien
- Benutzeroberflächen erstellen
- Fehleranalyse und Problemlösung
- Nutzung von GitHub zur Versionsverwaltung
